# REPORTER

Owner: Reporter

Use this file to compare planned work against executed work and calculate completion.

## What goes here
- Proposed vs executed
- Planned vs completed
- Completion percentage
- Blockers
- Dependencies
- Risk notes
- Status updates

## What does not go here
- Architecture decisions
- Code
- Permanent memory content

## Entry format
- ID
- Date
- Topic
- Planned
- Executed
- Completion %
- Blockers
- Next action
