# LUNA Context

Repositório oficial da documentação arquitetural da LUNA.

## Objetivo
Centralizar princípios, decisões arquiteturais e contexto global do organismo, mantendo a documentação técnica específica em cada repositório.

## Estrutura
- LUNA_CONSTITUTION.md
- LUNA_CONTEXT.md
- ECOSYSTEM_ARCHITECTURE.md
- CHANGELOG.md
- ADR/
- ORGANS/
- CHECKPOINTS/

## Regra de migração
1. Inventariar.
2. Copiar a documentação global.
3. Validar referências.
4. Somente então considerar remover documentos da origem.
5. Documentação específica permanece no repositório correspondente.
